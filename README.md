1. Run npm install
2. Run via UI, Navigate to the workspace and run "cypress open" 
3. Run the scripts via command prompt 
    -Navigate the project explorer
    -Type the command "cypress run --browser chrome --headed"
4. Once the execution is completed, navigate to the reports folder and open the mocha folder and the json mocha file to see the result
    -Screenshot in the "cypress/reports/mochareport/assets"
    -Video in the 'cypress/videos'
